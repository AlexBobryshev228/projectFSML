# projectFSML

Календарь-дневник (аналог TickTick по идее) для планирования задач, помодоро, привычек и целей недели.  
Проект для учебной работы: C++23 + Qt 6 + CMake + Git + тесты.

## Функционал (MVP)
- ✅ Задачи: создание/редактирование/удаление, статус выполнено, привязка к дате
- ✅ Недельный план (календарный вид)
- ✅ Помодоро: 25/5, старт/пауза/сброс, статистика сессий
- ✅ Привычки: чек-ин по дням, streak
- ✅ Цели недели: список целей + прогресс

> Примечание: синхронизация, уведомления и сложные повторения задач в MVP не входят.

## Технологии
- C++23
- Qt 6 (Qt Quick / QML)
- CMake
- SQLite (планируется/используется для хранения данных)
- GoogleTest (юнит-тесты на слой логики/данных)

## Сборка и запуск

### Qt Creator (рекомендуется)
1. File → Open File or Project… → выбрать `CMakeLists.txt`
2. Выбрать Kit (macOS Qt 6.x)
3. Run ▶

### Через терминал (CMake)
```bash
cmake -S . -B build
cmake --build build
./build/appProjectFSML   # имя может отличаться на вашей ОС
